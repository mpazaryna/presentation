# Marp Presentations

Standalone, self-contained presentations using MARP.
Each script is independently executable with zero dependencies between scripts.

## Available Scripts

### `scripts/apple-foundation-model.sh`
**When to use:** Build apple foundation model presentation

### `scripts/jstd.sh`
**When to use:** Build jstd presentation

### `scripts/mlx.sh`
**When to use:** Build mlx presentation

## Architecture

- **Self-Contained:** Each script includes all necessary code
- **No External Dependencies:** HTTP client embedded in each script
- **Progressive Discovery:** Only see what you need

## Quick Start

No authentication required for any script.
